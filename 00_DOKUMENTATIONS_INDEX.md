## 🔄 Konvertierung zu Word

### Methode 1: Automatische Konvertierung
```

pandoc 02_SPRINT_PLANNING_REVIEWS.md -o 02_SPRINT_PLANNING_REVIEWS.docx

```

### Methode 2: Manuell in Word
1. Neue Word-Datei öffnen
2. Inhalt kopieren (Markdown → Word formatieren)
3. Überschriften, Tabellen, Formatierung anpassen
4. Speichern als .docx

- https://pandoc.org/

## 📝 Markdown zu Word Formatierung

### Headers
```markdown
# Title          → Heading 1
## Subtitle      → Heading 2
### Section      → Heading 3
```

### Tabellen
```markdown
| Col 1 | Col 2 |  → Wird zu Word-Tabelle konvertiert
|-------|-------|
| Val 1 | Val 2 |
```

### Checklisten
```markdown
- [x] Done      → ☑ Done
- [ ] Todo      → ☐ Todo
```

### Code
````markdown
```csharp
Code hier...
```
→ Code Block in Word mit Syntax-Highlighting
````
